# 의료 데이터 분석 첫걸음 — Amyloid PET 기반 학습 프로젝트

## 소개
서울아산병원 핵의학과에서 방사성의약품 임상시험 및 FBB(아밀로이드)·SMBT-1(타우/성상교세포) PET/CT 트레이서를 활용한 알츠하이머 및 신경퇴행성 뇌질환 임상시험에 참여하며, 다양한 임상 데이터가 통합적으로 수집·관리되고 연구에 활용되는 과정을 가까이서 지켜보았습니다. 이 과정에서 의료영상 데이터가 어떻게 임상적 판단으로 이어지는지 직접 분석해보고 싶다는 관심이 생겼고, 이를 계기로 시작하게 된 학습 프로젝트입니다.

## 사용 데이터
- 현재 단계에서는 개인정보 보호를 위해 가상으로 생성한 연습용 데이터를 사용합니다.
(실제 환자 데이터 아님 — 아밀로이드 PET SUVR, MMSE 등 임상연구에서 흔히 쓰이는 변수 구조를 참고하여 생성)
- 추후 공개 데이터셋(예: ADNI 공개 샘플, Kaggle Alzheimer's Dataset)으로 확장 예정입니다.

- [ ] 공개 데이터셋으로 동일 분석 반복
- [ ] 간단한 분류 모델 (scikit-learn) 적용
- [ ] 결과 해석 및 임상적 의미 정리

## 배경 지식 / 강점
- 의료기관(서울대병원, 고려대병원, 가톨릭대병원, 삼성서울병원, 서울아산병원)에서 연구원으로 근무하며
  임상시험 데이터 수집·관리, IRB 심의, 임상연구윤리 준수 경험 보유
- 방사선학부 졸업, 현재 컴퓨터공학 학사 과정 병행 중 (2027년 상반기 졸업 예정)
